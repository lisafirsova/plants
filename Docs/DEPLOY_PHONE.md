# Установка Plants на настоящий телефон

Мобильное приложение больше не подключается к PostgreSQL напрямую. Все данные и
запросы ИИ проходят через единый сервер `Plants.AiProxy`.

## 1. Разместить PostgreSQL

Создайте постоянную PostgreSQL-базу у облачного провайдера и примените файл:

```text
Database/schema_3nf_postgresql.sql
```

Сохраните выданную строку подключения. На сервере она задаётся переменной
`DATABASE_URL`. Пароль базы не должен попадать в Android-проект.

## 2. Разместить Plants.AiProxy

Сервер собирается Docker-файлом:

```text
Backend/Plants.AiProxy/Dockerfile
```

При создании Docker Web Service укажите:

```text
Dockerfile: Backend/Plants.AiProxy/Dockerfile
Health check: /health
```

Добавьте секретные переменные окружения:

```text
DATABASE_URL=строка подключения PostgreSQL
GEMINI_API_KEY=бесплатный ключ Google AI Studio
GEMINI_MODEL=gemini-2.5-flash
PLANTS_APP_KEY=длинная случайная строка
```

Ключ Gemini создаётся на странице `https://aistudio.google.com/apikey`.
Платёжная карта для бесплатного уровня Gemini API не требуется. Бесплатный
уровень имеет ограничения по количеству запросов.

Для `PLANTS_APP_KEY` используйте строку длиной не менее 32 случайных символов.
После публикации проверьте:

```text
https://ВАШ-СЕРВЕР/health
```

Ответ:

```json
{"status":"ok"}
```

Чтобы сервер не засыпал и ИИ отвечал без минутной задержки после простоя,
используйте постоянно работающий платный экземпляр. Бесплатные экземпляры
некоторых хостингов останавливаются при отсутствии запросов.

## 3. Собрать APK

В PowerShell:

```powershell
cd "C:\Users\user\Documents\New project 2"

powershell -ExecutionPolicy Bypass -File .\Scripts\build-phone.ps1 `
  -ApiBaseUrl "https://ВАШ-СЕРВЕР" `
  -Configuration Debug
```

Скрипт безопасно запросит `PLANTS_APP_KEY` в консоли без отображения введённых
символов, а затем выведет полный путь к APK. Передайте APK на телефон, разрешите установку
из выбранного источника и установите приложение.

Для публикации в Google Play необходимо создать release-keystore и собрать
подписанный пакет Release. Для личной установки и защиты курсового проекта
достаточно Debug APK.

## 4. Проверка

1. Отключите телефон от компьютера.
2. Оставьте только мобильный интернет.
3. Войдите через Google.
4. Добавьте растение.
5. Откройте `Справочник -> ИИ-консультант`.
6. Отправьте текстовый вопрос и фотографию.

Если эти действия работают без компьютера, сервер и приложение настроены
правильно.
